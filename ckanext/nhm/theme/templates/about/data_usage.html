    Licensing

    As a public institution we publish all data under a [TBC] license to encourage the data submission and exchange. However, other licenses may be used for some datasets, for example a specific licence will be used when the Museum is not the sole copyright owner.

    Learn more about the [Museum’s data policies].

    Attribution

    If you use our data, we encourage you to cite or attribute this work as you would any other scientific research (even if the license does not implicitly require you to). Many Museum datasets are the result of a huge amount of time and effort invested by Museum staff and collaborators. Some represent an individual’s work over many decades. Citing these data allows us to measure the impact of the research and encourages others to deposit their data through the portal.

    Preferred citation format

    All datasets are assigned a DOI. Please use this in all citations.

    Scott, B., & Smith , V. Dataset title Natural History Museum. http://doi:1.2.3.4/56789

    Data download

    Datasets can be downloaded via the download button on the dataset page. The portal will process download requests, package up the dataset, and email a link to the newly create data package. The downloadable data format is dependent on the data type of each dataset. For the collections dataset this will be as a Darwin Core Archive.
